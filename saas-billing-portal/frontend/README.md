## SaaS Billing Portal

A modern full-stack SaaS Billing Portal built using React, Vite, Supabase, and Recharts.

## Features include:

-User Authentication (Login/Register)
-Dashboard Analytics
-Invoice Management
-Billing System
-Settings Management
-Modern Dark SaaS UI
-Supabase Database Integration


Tech Stack

## Frontend:

-React + Vite
-Plain CSS
-Recharts
-React Router DOM

## Database:
-Supabase (PostgreSQL)

## Backend
-Node.js
-Express
-REST API architecture


## backend
cd backend
npm install
node server.js

## frontend
cd frontend
npm install
npm run dev
